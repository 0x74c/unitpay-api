[UnitPay](https://help.unitpay.ru/) API wrapper for Node.js

Documentation https://help.unitpay.ru and https://help.unitpay.money

## Installation

```sh
$ npm install unitpay-api
```

### Initialization

```typescript
import UnitPay from 'unitpay-api'

const unitpay = new UnitPay({
	domain: 'unitpay.ru',
	secretKey: 'secretKey',
})
```
